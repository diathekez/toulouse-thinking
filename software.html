Recursos: Software

Algunos programas de software que usamos que recomendamos encarecidamente.

Productividad

Everything

voidtools.com

Una aplicación de búsqueda que puede encontrar cualquier archivo o carpeta almacenada en tu computadora en unos pocos segundos o menos. El programa también proporciona varios filtros, métodos de clasificación y una interfaz limpia y sencilla. La función de búsqueda funciona casi instantáneamente, en comparación con el método predeterminado de búsqueda de archivos de tu computadora.

Notion

notion.so

Un software de gestión de proyectos y toma de notas. Está diseñado para ayudar a los miembros de un equipo a coordinar plazos, objetivos y asignaciones en aras de la eficiencia y la productividad. También hay una versión de aplicación móvil, para que puedas mantener tus notas constantemente sincronizadas.

Nota: Notion aún no está disponible en español y tiene una curva de aprendizaje algo alta. Pero puedes copiar nuestra plantilla de notas de clase que traducimos al español.

Obsidian

obsidian.md

Obsidian es una poderosa base de conocimientos sobre una carpeta local de archivos Markdown de texto sin formato. Para decirlo en términos simples que lo harán relevante: Todas las computadoras del mundo pueden y siempre podrán abrir archivos Markdown. Y dado que tus archivos se almacenarán localmente, puedes decidir cómo hacer una copia de seguridad (backup) de tus notas.

No tienes que usar Obsidian para escribir en texto sin formato, pero Obsidian tiene herramientas, como una vista de gráfico que muestra todas tus notas vinculadas, que hacen que esto sea mucho más fácil y útil. Si lo aprendes a usar bien, Obsidian es casi como tener un segundo cerebro.

Nota: Obsidian aún no está disponible en español, pero puedes encontrar tutoriales en YouTube que te enseñan cómo usarlo. También hay una versión móvil, pero como todo se hace en el almacenamiento local, tendrás que sincronizar manualmente tus notas.

PowerToys

docs.microsoft.com/es-es/windows/powertoys

Este programa tiene un conjunto de herramientas que te permiten mantener las ventanas siempre visibles, ver los valores RGB y hexadecimales exactos de cualquier color que estes viendo, etc. Note: Solo para Windows.

Estudio

Anki

apps.ankiweb.net

Un programa gratuito y de código abierto de tarjetas didácticas que utiliza la repetición espaciada, una técnica de la ciencia cognitiva para una memorización rápida y duradera. Anki es mucho más eficiente que los métodos de estudio tradicionales; puede disminuir en gran medida el tiempo que dedicas a estudiar o aumentar en gran medida la cantidad que aprendes.

Si vas a probar alguna de estas recomendaciones, que sea Anki. Lo uso todos los días para estudiar y recordar conceptos y métodos de programación. Hay toneladas de tutoriales en YouTube sobre cómo usar Anki y crear tus propios mazos.

Herramientas Generales

qBittorrent

qbittorrent.org

qBittorrent es un cliente BitTorrent gratuito y de código abierto multiplataforma. Un cliente BitTorrent es un programa diseñado para compartir archivos entre pares (P2P) utilizando el protocolo BitTorrent. Lo que esto significa es que cuando descargas algo, no lo descargas de un servidor sino de otros usuarios como tu. Para descargar archivos con este protocolo, necesitas un archivo torrent, que es un archivo que contiene metadatos y rastreadores de red de esos archivos.

The Pirate Bay es la página web más popular para compartir archivos entre pares. De ninguna manera te animamos a que visites esta página, ya que la gran mayoría de estos archivos implican piratería. Pero si visitas esta página, te recomendamos que tengas cuidado al descargar software.

PowerISO

poweriso.com

Una potente herramienta de procesamiento de archivos de imágenes de CD/DVD/BD que te permite abrir, extraer, grabar, crear, editar, comprimir, cifrar, dividir y convertir archivos ISO, y montar archivos ISO en una unidad virtual interna.

Necesitarás al 100% este programa si planeas descargar software usando qBitTorrent. Recomendamos WinCDEmu como una alternativa de código abierto que es completamente gratuita

Malwarebytes

es.malwarebytes.com

Un escáner que analiza y elimina el software malicioso, incluido el software de seguridad no autorizado, el adware y el spyware. Viene con una versión de prueba gratuita de 14 días, pero solo debes ejecutarlo unas pocas veces y luego desinstalarlo. Y luego hazlo de nuevo cada seis meses más o menos.

La razón por la que tenemos una sección dedicada al mantenimiento de computadoras y la descarga de software es porque entendemos que la mayoría de nosotros no tenemos el dinero para poder comprar licencias y pagar subscripciones de software. No estamos defendiendo la piratería de ninguna manera; solo estamos diciendo que entendemos por qué podrías piratear software y queremos que tengas cuidado.

Wise Disk Cleaner

wisecleaner.com/wise-disk-cleaner.html

Elimina basura de tus navegadores, archivos inútiles y puede desfragmentar tus unidades de disco. Puedes programar limpiezas automáticas semanales o hacerlo manualmente.

Diseño Gráfico

Adobe

adobe.com/pe

Vamos a agrupar todo el paquete de software de edición de Adobe. No creeemos que estos programas necesiten una explicación, pero si no lo sabes, Photoshop, Illustrator, InDesign, Premiere Pro, etc. son software de edición estándar de la industria que deberías saber usar al menos a un nivel básico.

El software de Adobe es conocido por ser costoso. Si bien existen alternativas gratuitas como Photopea, GIMP y DaVinci Resolve, no son un reemplazo exacto. No estamos tolerando ni alentando la piratería, pero si encuentras un enlace que contiene versiones "cracked" de software costoso, simplemente te recomendemos que tengas cuidado y ejecutes Malwarebytes después de la instalación.

Figma

figma.com

Un editor de gráficos vectoriales y una herramienta de creación de prototipos que se basa principalmente en la web, con funciones adicionales fuera de línea habilitadas por aplicaciones de escritorio para macOS y Windows. Figma se enfoca en la interfaz de usuario y el diseño de la experiencia del usuario, con énfasis en la colaboración en tiempo real.

Si bien aún no está disponible en español, te recomendamos que aprendas los conceptos básicos de Figma usando tutoriales en YouTube porque si planeas un futuro en UX/UI, inevitablemente tendrás que aprender a usar Figma.

Audiovisual

Da Vinci Resolve

blackmagicdesign.com/es/products/davinciresolve

Recomendamos este programa sobre Adobe Premiere Pro si eres un completo principiante en el software de edición y/o quieres una interfaz más simple. Da Vinci Resolve es completamente gratuito y probablemente sea el único programa de edición de video que necesitarás.

Audacity

audacityteam.org

Un editor de audio digital gratuito y de código abierto y software de aplicación de grabación. Hay cientos de plugins disponibles para Audaicty, por lo que lo más probable es que sea el único editor de audio que necesitarás.

OBS Studio

obsproject.com/es

Un programa gratuito y de código abierto para screencasting y transmisión en vivo. OBS Studio proporciona captura en tiempo real, composición de escenas, grabación, codificación y transmisión a través del protocolo de mensajería en tiempo real (RTMP). Puedes transmitir videos a cualquier destino compatible con RTMP, incluyendo YouTube, Twitch, Instagram y Facebook.

El uso más relevante es que puedes grabar tu escritorio, tu cámara y tu micrófono con OBS Studio. Así que si tienes que hacer una presentación o estás trabajando en un proyecto que requiere evidencia de tu proceso, este programa es la mejor solución.

Handbrake

handbrake.fr

Una herramienta para convertir video de casi cualquier formato a una selección de códecs modernos y ampliamente compatibles. El aspecto más útil de Handbrake es que puedes comprimir tus archivos de video sin perder calidad. Simplemente juega con los ajustes de configuración hasta que encuentres el perfecto para ti.

Estética Visual

Rainmeter

rainmeter.net

Una utilidad de personalización de escritorio gratuita y de código abierto para Windows. Permite a los usuarios crear y mostrar widgets o applets de escritorio personalizables generados por el usuario llamados "skins" que muestran información. Las colecciones de skins listas para usar se pueden descargar e instalar en paquetes conocidos como "suites".

No hay una versión en español de Rainmeter, pero es un programa muy simple y si tienes dificultades, siempre puedes encontrar un tutorial en YouTube. También hay sitios web como Rainmeter Skins donde puedes encontrar suites listas para usar.

Wallpaper Engine

wallpaperengine.io/es

Una aplicación para Windows que permite a los usuarios utilizar y crear fondos de pantalla animados e interactivos. Necesitarás instalar Steam para usarlo y cuesta 11,95 soles. Hay miles de fondos de pantalla que puedes descargar e incluso crear los tuyos propios.
